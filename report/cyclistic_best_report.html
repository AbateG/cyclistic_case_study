Cyclistic Case Study

Author: Daniel Abate Garay
Role: Technical Solutions Architect & Data Scientist
Project Type: Exploratory Data Analysis + Strategic Recommendations
Tools Used: Python, R, Pandas, ggplot2, Seaborn, Markdown

Objective

Analyze Cyclistic's bike-share trip data to uncover behavioral differences between casual riders and annual members, and recommend strategies to increase membership conversion.

Project Structure

  • data/: Raw trip data files
  • scripts/: Python and R analysis scripts
  • outputs/: Cleaned data, charts, and summary tables
  • visuals/: Branded diagrams and project maps

Key Outputs

  • Ride length distribution by user type (2019-2020)
  • Ride counts by day of week (1=Sunday)
  • Top stations for casual vs. member riders
  • Strategic recommendations for membership growth

Data Sources

  • Divvy_Trips_2019_Q1.csv: 365,069 rows of Q1 2019 trip data
  • Divvy_Trips_2020_Q1.csv: 426,887 rows of Q1 2020 trip data

Analysis Summary

  • Members take shorter rides (mean ~12 min) primarily on weekdays (mode day 3-5), indicating commuting behavior.
  • Casuals take longer rides (mean ~40-44 min) more on weekends (mode day 1 or 7), suggesting leisure use.
  • Statistical tests confirm significant differences (p-value < 0.0001) in ride lengths between user types for both years.
  • Ride counts increased from 2019 to 2020, with members dominating volume.

Visualizations

  • Ride counts by day of week and user type
  • Ride length distributions
  • Average ride length by day
  • Ride counts by year

Recommendations

  1. Targeted Weekend Promotions for Casuals: Focus on tourist stations (Streeter Dr & Grand Ave, Lake Shore Dr & Monroe St, Shedd Aquarium) with weekend-only membership trials. Offer 2-week free trials during peak summer months to convert leisure riders.
  2. Weekday Member Retention: Enhance commuter benefits with priority docking at business districts, loyalty rewards for frequent weekday users, and discounted annual renewals for consistent riders.
  3. Digital Engagement Campaigns: Use app notifications to target casuals based on usage patterns - weekend reminders for membership benefits, personalized offers for frequent riders near tourist areas.
  4. Infrastructure Expansion: Add bike stations near tourist attractions and lakefront areas to capture casual growth. Increase weekday capacity in downtown business districts.
  5. Seasonal Marketing: Launch summer campaigns targeting casuals with family membership deals, while winter focuses on member retention with indoor cycling partnerships.
  6. Data-Driven Monitoring: Continue tracking post-pandemic trends, with quarterly analysis of user behavior changes and conversion rates.

Validation and Enhancements

  • Added row count assertions to ensure data integrity.
  • Capped ride lengths at 24 hours to handle outliers.
  • Implemented logging for better traceability.
  • Performed t-tests to validate statistical significance of differences.
