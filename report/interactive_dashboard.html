Dashboard Help Guide

This dashboard analyzes 3.1M+ bike trips from the Cyclistic bike-share program in Chicago. Use the tabs above to explore different aspects of the data.

  • Overview: Key metrics and summary charts
  • Temporal Analysis: Time-based patterns and trends
  • Maps & Routes: Geographic distribution of stations and routes
  • Data Explorer: Interactive table with full dataset

Use the filter controls at the top to focus on specific data segments:

  • User Type: Compare members (annual subscribers) vs casual users (pay-per-ride)
  • Time Period: Filter by year, weekdays vs weekends
  • Date Range: Optional specific date filtering

  • Box Plots: Display distribution statistics (e.g., ride lengths)
  • Maps: Geographic distribution of stations and usage

  • Pagination: Navigate through large datasets
  • Export: Download filtered data as CSV or JSON
