Objective: Analyze Cyclistic's bike-share trip data to uncover behavioral differences between casual riders and annual members, and recommend strategies to increase membership conversion.

Tools: Python, R, Pandas, ggplot2, Seaborn, Matplotlib

2. Data Sources & Preparation

  • Divvy_Trips_2019_Q1.csv: Q1 2019 trip data (pre-pandemic)
  • Divvy_Trips_2020_Q1.csv: Q1 2020 trip data

Data was cleaned, standardized, and merged. Calculated fields include ride_length (duration in minutes) and day_of_week (1=Sunday, 7=Saturday). Outliers and invalid records were removed.

import pandas as pd
from scipy.stats import ttest_ind

# Statistical analysis
member_duration = df[df['member_casual'] == 'member']['ride_length']
casual_duration = df[df['member_casual'] == 'casual']['ride_length']

t_stat, p_value = ttest_ind(member_duration, casual_duration)
print(f"T-test p-value: {p_value:.2e}")

library(tidyverse)

# Data analysis pipeline
df %>%
  group_by(member_casual) %>%
  summarise(
    avg_duration = mean(ride_length_minutes),
    total_rides = n(),
    .groups = "drop"
  ) %>%
  arrange(desc(avg_duration))

SELECT
    member_casual,
    COUNT(*) as total_rides,
    ROUND(AVG(ride_length_minutes), 2) as avg_duration
FROM cyclistic_trips
GROUP BY member_casual
ORDER BY total_rides DESC;
